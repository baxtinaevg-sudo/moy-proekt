# Управление доступом к moy-proekt

## Репозиторий
- **URL:** https://github.com/baxtinaevg-sudo/moy-proekt
- **Видимость:** PRIVATE (приватный)
- **Владелец:** baxtinaevg-sudo

---

## Текущий статус доступа

| Участник | Роль | Права |
|----------|------|-------|
| baxtinaevg-sudo | Admin | Полный доступ (admin, maintain, push, triage, pull) |

**Коллаборантов кроме владельца — нет.**

---

## Как дать доступ другому человеку

1. Открой https://github.com/baxtinaevg-sudo/moy-proekt/settings/access
2. Нажми **"Add people"**
3. Введи имя пользователя GitHub или email
4. Выбери уровень доступа:
   - **Read** — только чтение
   - **Triage** — чтение + управление issues
   - **Write** — чтение + запись (push)
   - **Maintain** — управление проектом без удаления
   - **Admin** — полный доступ
5. Человек получит приглашение и должен его принять

## Как отозвать доступ

1. Открой https://github.com/baxtinaevg-sudo/moy-proekt/settings/access
2. Найди человека в списке
3. Нажми **"Remove"**
4. Доступ будет отозван немедленно

## Как поделиться ссылкой

⚠️ **Важно:** Ссылка на приватный репозиторий **НЕ даёт доступа** никому, кроме коллаборантов. Если ты отправишь ссылку `https://github.com/baxtinaevg-sudo/moy-proekt` кому-либо — они увидят только страницу "404 Not Found", если не добавлены в коллабораторы.

### Способы поделиться кодом:
1. **Добавить в коллабораторы** — человек получит доступ ко всему репозиторию
2. **Создать fork** — невозможен для приватного репозитория без доступа
3. **Отправить файлы вручную** — скачать архив и передать
